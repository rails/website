<p>Welcome to the third edition of This Week in Rails, a weekly report with highlights from the Rails community. My apologies for the delay of this post, the past two weeks have been pretty crazy, so this edition covers the most interesting articles and news from the past two weeks.</p>
<p>Let&#8217;s kick off this report with a couple of maintenance releases by Jamis Buck. Both <a href="http://weblog.jamisbuck.org/2008/6/28/capistrano-2-4-3">Capistrano 2.4.3</a> and <a href="http://weblog.jamisbuck.org/2008/6/28/net-ssh-2-0-3">Net::<span class="caps">SSH</span> 2.0.3</a> were published two weeks ago. If you use them, consider upgrading.</p>
<p>Rails 2.1 has been out for a while now, but in case you didn&#8217;t have a chance to catch up yet, <a href="http://jasonrudolph.com/blog/2008/07/08/5-resources-for-getting-up-to-speed-on-rails-21/">this post</a> collects several links to useful resources which will help bring you up-to-date.</p>
<p>The Pathfinder Development&#8217;s blog put out three highly interesting posts. The first is <a href="http://www.pathf.com/blogs/2008/06/more-named-scope-awesomeness">More Named Scope Awesomeness</a> by Noel Rappin, while the second and third ones are <a href="http://www.pathf.com/blogs/2008/07/pretty-blocks-in-rails-views/">Pretty blocks in Rails views</a> and <a href="http://www.pathf.com/blogs/2008/07/drying-up-rails-controllers-polymorphic-and-super-controllers/">DRYing up Rails Controllers: Polymorphic and Super Controllers</a>, both by Josh Symonds. Another good (and quick) recent read about controllers, was &quot;MVC: How to write controllers &quot;:http://andrzejonsoftware.blogspot.com/2008/07/mvc-how-to-write-controllers.html.</p>
<p>The same Noel also published the second part of &#8220;Developing iPhone applications using Ruby on Rails and Eclipse&#8221; for DeveloperWorks (<a href="http://www.ibm.com/developerworks/opensource/library/os-eclipse-iphoneruby1/">part 1</a><br />
 and <a href="http://www.ibm.com/developerworks/opensource/library/os-eclipse-iphoneruby2/">2</a>).</p>
<p>FiveRuns released a valuable gem called <a href="http://blog.fiveruns.com/2008/7/9/introducing-data_fabric">data_fabric</a> which adds support for sharding and replication to Active Record. The same company also <a href="http://blog.fiveruns.com/2008/7/10/run-your-way-to-railsconf-europe">has a contest up</a> and they&#8217;re offering two free tickets to RailsConf Europe in Berlin. Speaking of conferences, Fabio Akita announced that there will be a <a href="http://akitaonrails.com/2008/7/11/rails-summit-brazil-2008">Rails Summit Brazil 2008</a> this coming October in SÃ£o Paulo. This will be the first event of its kind for the Rails community in South America.</p>
<p><a href="http://blog.rayapps.com/2008/06/28/activerecord-oracle-enhanced-adapter-version-111-released/">An improved version</a> (i.e. 1.1.1) of the Oracle enhanced adapter was released, as well as <a href="http://rubyforge.org/frs/?group_id=2361&amp;release_id=23307">version 0.9.5</a> of the IBM_DB adapter for DB2 and Informix, which adds support for Rails 2.1.</p>
<p>In purely chronological order, I found the following articles to be worth pointing out: <a href="http://www.plopcentral.com/articles/2008/07/02/rails-development-environment-slow-in-vista">Speed up slow Rails development in vista</a> &#8211; a handy tip for developers using Vista, <a href="http://www.developer.com/open/article.php/10930_3757576_1">Adding Google Maps To Your Rails Applications</a>, <a href="http://blog.zmok.net/articles/2008/07/10/live-fulltext-search-in-ruby-on-rails">Live fulltext search in Ruby on Rails</a> and <a href="http://rubypond.com/articles/2008/07/11/useful-flash-messages-in-rails/">Useful Flash Messages in Rails</a>.</p>
<p>The Railscasts website published two new episodes, one on <a href="http://railscasts.com/episodes/116">testing through Selenium</a>, and another on <a href="http://railscasts.com/episodes/117">semi-static pages</a>.</p>
<p>Finally, let&#8217;s close this edition on a lighter note. The next time you are about to create an acts_as_an_evil_genius plugin or other imaginatively named one, think about <a href="http://www.pluginaweek.org/2008/07/06/acts_as_what/">this post</a>. ;-)</p>
<p>If you&#8217;d like to read more updates from the Ruby side of things, please head over to <a href="http://antoniocangiano.com/2008/07/11/this-week-in-ruby-july-11-2008">This Week in Ruby</a>.</p>
